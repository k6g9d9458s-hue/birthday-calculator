Birthday Calculator

Описание проекта

Программа на Python, которая:

* запрашивает день, месяц и год рождения пользователя;
* определяет день недели, на который приходится дата рождения;
* определяет, является ли год рождения високосным;
* вычисляет текущий возраст пользователя;
* выводит дату рождения в формате электронного табло с использованием символов *.

Используемые технологии

* Python 3
* Модуль datetime

Запуск программы

1. Запустите файл main.py.
2. Введите день, месяц и год рождения.
3. Получите результаты работы программы.
